# Spring boot 2 sample in Kotlin

Spring Boot 2.0.3, Kotlin 1.2.51

## Preparation

Please create Database and Table.

```sql
CREATE TABLE task(
   id        SERIAL PRIMARY KEY,
   name      VARCHAR,
   created_at TIMESTAMP,
   updated_at TIMESTAMP
);
```
